User Profiles stored on the blockchain.

Each entry will add a User record to the Ethereum blockchain. It will store the senders address, their username, and a hash that references an endpoint on the IPFS network. AKA the Interplanetary File System. This was my final project for Consensys Academy which I took in hopes of learning the basics of Solidity smart contract develpoment, and building in the Web3 ecosystem.
